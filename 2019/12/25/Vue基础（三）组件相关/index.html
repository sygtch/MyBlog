Vue基础（三）

安装vue

  • 安装npm：在nodejs.org下载nodejs（用node -v和npm -v检查；另可用npm install -g npm升级npm）
  • 使用淘宝镜像（略过）$ npm install -g cnpm –registry=https://registry.npm.taobao.org
  • 安装vue-cli : cnpm install vue-cli -g //全局安装 vue-cli

安装后，选定文件夹，执行

npm init webpack"project-name"

4. 组件

  • 初识组件：在app.js文件中

    Vue.component("gtc",{
    template:'
        <p>
            {{name}} : 哈哈哈哈哈
            <button @ckick="change" >更改</button>
        </p>',                //template只能有一个根标签，
     data:function(){
         return{
             name: "初始"
         }
     },
    methods:{
        change:function(){
            this.name = "改变了";
        }
    }
})

  • 建立自己的组件：

    1. 在components文件夹下建立Users.vue
    <template>
    <div class="users">
        <ul>
            <li v-for="user in users">
            {{user}}
            </li>
        </ul>
    </div>
</template>

<script>
export default{
        name:"users",
        data(){
            return{
                users:["Herry","Bucky","Jim"]
            }
        }
    }
</srript>

<style scoped>
    //加了scoped后，可以控制样式的作用范围，不会发生冲突或继承
</style>

    1. 注册组件：让我们的组件可以被使用

      2.1 在main.js中进行，全局注册组件（不推荐全局）

      ...
import Users from './components/Users'

Vue.component("users",Users);

      使用时：在App.vue中可以直接使用标签

      2.2 局部，在App.vue中

      ...


 <script>

 import Users from './components/Users'

 export default{
         name:"users",
         data(){
             return{

             }
         },
         components:{
             "users":Users    //此处users可省略，起名时不能为关键字
         }
     }
 </srript>
 ```

    2. 子组件向父组件传值：效果为，子组件间有相同的依赖元素，点击某一子组件，则所有依赖于该元素的值都发生改变；
