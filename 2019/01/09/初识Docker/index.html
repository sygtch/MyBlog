Docker上搭建自己的Git管理平台

初识docker

常用命令

# 查找centos镜像
docker search centos
# 下载centos镜像
sudo docker pull docker.io/centos:latest

# 查看镜像
sudo docker image ls
# 删除镜像
sudo docker image rm 123456789
# 若有容器(container)在引用此镜像会造成删除失败，则先删除容器
# 先查看正在运行的container，再根据container的id删除
docker ps -a
docker rm 123456(containerId)
docker image rm 123456789(imageId)

# 创建容器并进入其中，参数说明: -t分配一个伪终端并绑定到容器的标准输入，-i 让容器的标准输入保持打开
sudo docker run -t -i --name guo-centos docker.io/centos:latest /bin/bash
#  退出容器
exit
# 删除容器
sudo docker rm guo-centos

# 查看所有容器
sudo docker ps -a
# 查看当前运行的容器
sudo docker ps

# 启动容器
sudo docker start guo-centos
# 停止容器
sudo docker stop guo-centos
# 进入容器环境
sudo docker attach guo-centos

#容器内部发送指令
sudo docker exec guo-centos touch xyz.txt

高级命令

# 将guo-centos容器内的xyz.txt文件拷贝到根目录下
sudo docker cp guo-centos:xyz.txt ./

# 将自定义容器发布为镜像，参数说明：-a为作者，-m为提交信息
sudo docker commit -a "GTC" -m "add txt" guo-centos guo-centos:0.1

# 基于新建的镜像创建容器，生成的新容器，包含镜像内曾经有的xyz.txt
sudo docker run -t -i --name another-centos guo-centos:0.1 /bin/bash

# 查看镜像信息
sudo docker history guo-centos:0.1
# 查看容器内变动日志
sudo docker diff guo-centos  
# 查看镜像的详细信息
sudo docker inspect zhaoolee-centos:0.1 
# 查看容器的详细信息
sudo docker inspect zhaoolee-centos

私有库的创建

# 1.下载私有仓库注册服务器的镜像
sudo docker pull registry:latest

# 2.创建一个注册服务器容器,参数说明 -d容器在后端运行, -p 5000:5000在容器的5000端口运行并映射到外部系统的5000端口, --name server-registry容器命名为guo-server-registry, -v /tmp/registry /tmp/registry把宿主机的目录/tmp/registry挂载到容器目录/tmp/registry
sudo docker run -d -p 5000:5000 --name guo-server-registry -v /tmp/registry:/tmp/registry docker.io/registry:latest

# 3.为本地镜像添加标签，并将其归入本地仓库
sudo docker tag guo-centos:0.1 localhost:5000/guo-centos:0.1
# 将被标记的本地镜像推到库中
sudo docker push localhost:5000/guo-centos:0.1

# 4.使用
sudo docker pull 143.xxx.xxx.xxx:5000/guo-centos:0.1

卸载docker

如果已有老版本docker，先查看已安装的docker：

yum list installed | grep docker

可以看见已经安装的：

将其卸载，以防冲突：

yum -y remove  containerd.io.x86_64  docker-ce.x86_64  docker-ce-cli.x86_64

安装docker

# 通过yum源安装docker
yum -y install docker
# 启动docker
systemctl start docker
# 开机自启
sudo systemctl enable docker

也可以通过“宝塔”进行快速安装设置。

下载gogs的镜像

docker pull gogs/gogs

创建gogs的储存文件夹

mkdir -p /var/gogs

运行镜像，产生容器，映射端口

docker run --name=gogs -p 10022:22 -p 10080:3000 -v /var/gogs:/data gogs/gogs
